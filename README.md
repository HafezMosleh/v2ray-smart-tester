## 🌟 Overview
Stop manually checking your proxies! This intelligent script automatically pings and benchmarks your V2Ray/Xray configurations. It bypasses censorship hurdles by quickly identifying dead nodes and sorting the remaining ones by real latency, ensuring you always have the fastest connection possible.

## ✨ Key Features
- 🚀 **Blazing Fast Concurrent Testing**: Test hundreds of nodes in seconds.
- 🧹 **Auto-Cleanup**: Automatically removes blocked, timeout, or dead configurations.
- 📊 **Smart Sorting**: Orders your configs based on actual response time (ping).

## 🚀 Getting Started

### Prerequisites
Make sure you have `Python` installed on your local machine.

### Installation
```bash
git clone https://github.com/HafezMosleh/v2ray-smart-tester.git
cd v2ray-smart-tester
pip install urllib3
```

### Usage
```bash
python main.py <config-or-subscription-link>
```
